8. Даны переменные a и b. Проверьте, что a делится без остатка на b. Если это так — выведите «Делится» и результат деления, иначевыведите «Делится с остатком» и само значение деления.
Входные: a=10, b = 5 → Результат:Делится, результат = 2
Входные: a=14, b = 3 → Результат:Делится с остатком,остаток= 2
